# Sistema de Gestión de Biblioteca

Aplicación desarrollada en Java para administrar los préstamos de libros
de una biblioteca.

El sistema permite registrar socios, libros y préstamos, controlando
las reglas de préstamo para estudiantes y docentes.

## Tecnologías

- Java
- Programación Orientada a Objetos
- Colecciones genéricas
- Persistencia de datos

## Funcionalidades

- Registro de socios (estudiantes y docentes)
- Registro de libros
- Gestión de préstamos y devoluciones
- Control de préstamos vencidos
- Listado de socios y libros
